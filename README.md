# 📖 Gerador de Citações Bíblicas

Um gerador de versículos estruturado em **HTML5, CSS3 e JavaScript puro (ES6)**. O projeto foi desenhado inicialmente no Figma e desenvolvido com foco em uma interface limpa, rápida e com uma identidade visual litúrgica (tons de vinho e ouro velho).

---

## 🚀 Funcionalidades

* **Filtro por Temas:** Seleção de versículos baseada em categorias (Amor, Fé, Paz, Força, Sabedoria, etc.).
* **Sorteio Dinâmico:** Algoritmo que filtra o banco de dados local e sorteia um versículo correspondente via JavaScript (`Math.random`).
* **Mensagens de Reflexão:** Exibição de insights práticos acoplados aos versículos selecionados.
* **Interface Responsiva:** Design adaptável para desktop e dispositivos móveis.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do esqueleto do app.
* **CSS3:** Estilização avançada, transições suaves e paleta de cores personalizada.
* **JavaScript (ES6):** Manipulação do DOM, manipulação de arrays (`.filter`, `.map`) e lógica de sorteio.
